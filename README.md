# Ecommerce (Assessment)

This assessment is for CurlWare, a project built with Laravel 11, React 18, and MySQL.

## Installation

```bash
  composer install
```

```bash
  npm install
```

## Run Locally

```bash
  php artisan serve
```

```bash
  npm run dev
```

## API Reference

Access the API documentation at http://127.0.0.1:8000/request-docs
