# Ecommerce (Assessment)

This assessment is for CurlWare, a project built with Laravel 11, React 18, and MySQL.

## Installation

```bash
  composer install
```

```bash
  npm install
```

## Run Locally

```bash
  php artisan serve
```

```bash
  npm run dev
```
